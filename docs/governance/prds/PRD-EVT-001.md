# PRD-EVT-001: Event Builder & Public Page

## Description
Core engine for creating and rendering event pages.

## Features
- **Event CRUD:** Create, Read, Update, Delete events.
- **Section Toggles:** Enable/Disable blocks (Hero, Schedule, FAQ).
- **Template Selection:** Choose from "Centennial", "Revival", "Camp".
- **SEO:** Dynamic meta tags per event.
- **Preview Mode:** Real-time preview before publishing.
