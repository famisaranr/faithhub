# PRD-PWA-001: Installable Church App

## Description
Progressive Web App layer to allow "installation" on user devices.

## Features
- **Dynamic Manifest:** Generates `manifest.json` on the fly using Tenant branding.
- **Icon Generation:** Serves correct favicon/app-icon for the specific church.
- **Offline Shell:** Service Worker caches UI shell and recent content.
- **Install Prompt:** UI nudge to add to homescreen.
