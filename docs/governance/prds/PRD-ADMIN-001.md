# PRD-ADMIN-001: Admin Console

## Description
Central dashboard for church admins and platform super-admins.

## Features
- **Tenant Admin:** Manage church profile, branding, and billing.
- **Event List:** Dashboard of active/past events.
- **RSVP Exports:** Bulk actions for guest data.
- **Domain Status:** DNS propagation checker.
- **Roles:** Invite other admins/editors.
