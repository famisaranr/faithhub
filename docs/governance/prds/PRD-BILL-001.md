# PRD-BILL-001: Billing

## Description
Monetization and subscription management system.

## Features
- **Payment Gateway:** Integration with PayMongo (PH) and Stripe (Global).
- **Plan Gating:** Restrict features (e.g., PWA, Custom Domain) based on tier.
- **Trials:** 14-day free trial logic.
- **Coupons:** Discount code support for conferences.
- **Invoices:** Auto-generated PDF receipts.
