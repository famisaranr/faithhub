# PRD-RSVP-001: Guest Management

## Description
System for handling guest registrations and headcounts.

## Features
- **Dynamic Forms:** Configurable questions per event.
- **CSV Export:** Downloadable guest lists for clerks.
- **Capacity Limits:** Auto-close RSVP when full.
- **Attendance Status:** Going, Maybe, Not Going.
- **Group Size:** "How many are coming with you?"
